# ⚽ Power BI DataViz World Championship 2026

## Project Overview
Built a 5-page executive Power BI report for the *Power BI DataViz World Championship (FabCon Barcelona '26)*, analyzing PitchSide Pro's revenue performance across 8 countries and 3 macro-regions (EMEA, Americas, and APAC) from 2021–2025.

The report answers the key business question:

> *What is driving changes in revenue over time, and where should the business focus next?*

## Report Pages
- 📊 Executive Overview
- 📦 Product Analysis
- 👥 Customer Analysis
- 📈 Performance Over Time
- 🎯 Performance Drivers

## Key Metrics
- Total Revenue: *11.22M*
- ROI: *111.9%*
- Profit Margin: *52.81%*
- Customers: *29K*
- Average Order Value (AOV): *110.36*
- YOY Growth: *30.10%*
- MOM Growth: *2.27%*
